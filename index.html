🛠️ Build your own command

Custom commands are just skills: a SKILL.md file in .claude/skills/<name>/. The directory name becomes the command. Pull in live data with !`command` and accept input with $ARGUMENTS.

# ~/.claude/skills/summarize-changes/SKILL.md
---
description: Summarize uncommitted changes and flag risks.
---

## Current changes
!`git diff HEAD`

## Instructions
Summarize the changes above, then list any risks.
